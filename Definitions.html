Defenitions

File
A file is a logical collection of information stored under a single name.
Information
Information is a concise detail of data listed about a situation. words, pictures, and sounds which all have meaning to us.
Directory
Directory is a logical collection of files stored under a single name.
Jpeg
A format for compressing image files. JPEG stands for Joint Photographic Experts Group
PNG
Is a raster graphics file format that supports lossless data compression. PNG was created as an improved, non-patented replacement for Graphics Interchange Format (GIF), and is the most widely used lossless image compression format on the Internet.
GIF
A lossless format for image files that supports both animated and static images.
PDF
A file format that provides an electronic image of text or text and graphics that looks like a printed document and can be viewed, printed, and electronically transmitted.
